Garbage Clasification

The Garbage Classification using Arduino project aimed to automatically sort various types of garbage to address waste management challenges. As a key team member, I contributed to designing the system, implementing sensors, and programming classification algorithms. The Arduino microcontroller, along with ultrasonic, color, and infrared sensors, detected garbage characteristics for accurate classification. Machine learning algorithms were employed to differentiate recyclables, organic waste, and non-recyclables. Actuators directed sorted items to the appropriate bins. We continuously improved accuracy through testing and adjustments. The user-friendly interface facilitated real-time monitoring and interaction. This project positively impacted waste management, promoting recycling and minimizing environmental harm. It was a rewarding experience utilizing Arduino, sensors, and machine learning to address this societal issue.

Human Following Robot

The Human Following Robot project aimed to create an autonomous robot that could track and follow a human subject. Using cameras, sensors, and motion planning algorithms, the robot detected and tracked the person's movements while avoiding obstacles. Through extensive testing and optimization, we developed a reliable and user-friendly system. This project has potential applications in personal assistance, surveillance, and entertainment. It was a rewarding experience combining technical expertise to create a sophisticated human-following robot

Airport Trolley Rapid Prototyping

The Airport Trolley Rapid Prototyping project focused on creating functional airport trolleys using bamboo. We utilized rapid prototyping techniques to design and build lightweight and sustainable trolley prototypes. Bamboo proved to be a durable and eco-friendly material. The trolleys featured adjustable handles, secure storage, and sturdy wheels. This project demonstrated the potential of bamboo in revolutionizing airport trolley design.
